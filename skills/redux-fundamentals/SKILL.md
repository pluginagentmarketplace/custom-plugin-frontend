# Redux Fundamentals

Master the core concepts of Redux for building predictable, maintainable state management.

## MANDATORY
- Redux store creation and configuration
- Pure reducer functions and action creators
- Understanding dispatch and actions
- React-Redux integration (useSelector, useDispatch)
- Combining reducers with combineReducers
- Redux DevTools setup and debugging

## OPTIONAL
- Redux Thunk for async actions
- Custom middleware development
- Selector pattern for derived state
- Performance optimization with reselect
- Redux Toolkit basics and advantages
- Redux Saga introduction

## ADVANCED
- Middleware composition and chaining
- Complex async patterns
- Normalized state structure
- Time-travel debugging
- Error handling in reducers
- Testing Redux logic
