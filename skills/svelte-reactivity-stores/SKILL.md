# Svelte Reactivity & Stores

Compile-time framework with true reactivity.

## MANDATORY
- Reactive variables and declarations
- Two-way data binding
- Reactive statements ($:)
- Stores (writable, readable, derived)
- Component lifecycle
- Event handling

## OPTIONAL
- Custom store implementations
- Store subscriptions and auto-subscriptions
- Context API
- Transitions and animations
- Actions
- Component composition

## ADVANCED
- Store contract implementation
- Cross-component state patterns
- Performance optimization
- SSR with SvelteKit
- Custom transitions
- Testing Svelte components

## Assets
- See `assets/svelte-reactivity-config.yaml` for patterns
