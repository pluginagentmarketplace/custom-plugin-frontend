# Vite Modern Bundling

Experience next-generation build tooling with Vite.

## MANDATORY
- Native ES modules in development
- Vite project creation and setup
- Configuration with vite.config.ts
- Hot Module Replacement (HMR)
- Framework plugins (React, Vue, Svelte)
- Production builds with Rollup

## OPTIONAL
- Environment variables (.env files)
- CSS preprocessors (Sass, Less)
- Path aliases configuration
- Proxy configuration for APIs
- Multi-page applications
- Library mode builds

## ADVANCED
- SSR configuration
- Custom plugins development
- Build optimization strategies
- Pre-bundling configuration
- Worker threads support
- Advanced Rollup configuration

## Assets
- See `assets/vite-config.yaml` for configuration patterns
