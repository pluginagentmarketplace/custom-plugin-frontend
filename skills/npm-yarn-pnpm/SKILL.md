# Package Managers (NPM, Yarn, PNPM)

Master modern package managers for dependency management and task automation.

## MANDATORY
- Package installation and management
- package.json and lock files
- Semantic versioning (semver)
- npm scripts for automation
- Development vs production dependencies
- Global vs local installations

## OPTIONAL
- Workspaces and monorepo management
- Security auditing (npm audit)
- Package publishing to registries
- Private registries configuration
- Yarn PnP (Plug'n'Play)
- PNPM strict dependencies

## ADVANCED
- Custom npm scripts with pre/post hooks
- Package linking for local development
- Patch-package for dependency fixes
- Monorepo tools (Turborepo, Nx)
- Dependency version strategies
- Publishing scoped packages

## Assets
- See `assets/package-manager-config.yaml` for configuration patterns

- [Yarn Docs](https://yarnpkg.com/)
