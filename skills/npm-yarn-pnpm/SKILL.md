# Package Managers (NPM, Yarn, PNPM)

Master modern package managers for dependency management and task automation.

## Key Topics
- Package installation and versioning
- package.json and lock files
- Semantic versioning
- npm scripts and task automation
- Workspaces and monorepo management
- Security auditing
- Package publishing

## Learning Outcomes
- ✅ Install and manage dependencies effectively
- ✅ Understand package.json structure
- ✅ Use npm scripts for automation
- ✅ Manage dev vs production dependencies
- ✅ Compare NPM, Yarn, and PNPM
- ✅ Set up monorepos with workspaces
