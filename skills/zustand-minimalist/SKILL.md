# Zustand Minimalist

Lightweight, scalable state management library with minimal boilerplate.

## MANDATORY
- Store creation with Zustand
- State and actions definition
- useShallow hook for shallow equality
- Selectors for derived state
- Basic middleware (persist, devtools, immer)
- Subscriptions and listeners
- Testing stores

## OPTIONAL
- Persist middleware for storage
- Immer middleware for immutable updates
- DevTools middleware for debugging
- Custom middleware development
- Async actions and thunks
- Store composition and combining stores
- Performance optimization with selectors

## ADVANCED
- Complex middleware stacking
- Type-safe stores with TypeScript
- Large-scale store architecture
- Advanced async patterns
- State synchronization
- Server state management
- Cross-tab communication
