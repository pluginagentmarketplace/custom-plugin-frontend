# Skill: Browser DevTools Mastery

**Level:** Advanced
**Duration:** 1 week
**Agent:** Performance
**Prerequisites:** Fundamentals Agent

## Overview
Master browser DevTools for performance profiling, debugging, and optimization. Learn Chrome/Firefox developer tools thoroughly.

## Key Topics

- Performance panel
- Network analysis
- Memory profiling
- Coverage tool
- Lighthouse integration
- Custom metrics

## Learning Objectives

- Profile applications
- Identify bottlenecks
- Debug memory leaks
- Analyze network
- Measure custom metrics

## Practical Exercises

### Performance profiling
1. Open DevTools Performance tab
2. Start recording
3. Interact with page
4. Stop and analyze
5. Look for long tasks

### Network analysis
1. Open Network tab
2. Reload page
3. Analyze waterfall
4. Identify slow requests
