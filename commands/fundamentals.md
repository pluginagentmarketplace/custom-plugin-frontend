# /fundamentals

Learn frontend fundamentals: HTML, CSS, JavaScript, Git, DOM, and web basics.

## Usage

```
/fundamentals [skill-name]
```

## Available Skills

- internet-basics
- html-css-essentials
- dom-manipulation

## Examples

```
/fundamentals internet-basics
/fundamentals html-css-essentials
/fundamentals dom-manipulation
```

## Description

Start your frontend journey with core technologies. This agent guides you through:
- Web fundamentals and how the internet works
- HTML5 semantic markup and structure
- CSS layouts with Flexbox and Grid
- JavaScript programming basics
- DOM manipulation and events
- Git version control

Perfect for absolute beginners or those wanting to solidify fundamentals.
